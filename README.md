# MedKG-Core

A task-driven medical knowledge graph platform for multi-user collaboration, supporting graph construction and querying from structured, semi-structured, and unstructured medical data.

## Overview

**MedKG-Core** is a Java-based medical knowledge graph project designed to support collaborative graph construction, management, and querying.

The project focuses on medical data governance, medical annotation, clinical knowledge modeling, and healthcare knowledge graph applications. It provides a task-driven workflow that allows multiple users to participate in data processing, entity annotation, relation extraction, graph construction, and graph-based retrieval.

## Key Features

- **Task-driven workflow**
  - Manage graph construction tasks by project, dataset, user, and progress.
  - Support collaborative workflows for multi-user medical data processing.

- **Multi-source medical data support**
  - Structured data: database tables, forms, standard medical datasets.
  - Semi-structured data: Word, TXT, medical records with fixed fields.
  - Unstructured data: clinical notes, discharge summaries, medical documents, guidelines, and research text.

- **Medical knowledge graph construction**
  - Medical entity extraction.
  - Entity normalization.
  - Relation extraction.
  - Attribute modeling.
  - Ontology and schema management.
  - Graph data import and update.

- **Medical annotation support**
  - Entity annotation.
  - Relation annotation.
  - Task assignment.
  - Annotation quality control.
  - Review and correction workflow.

- **Graph query and retrieval**
  - Entity search.
  - Relation query.
  - Path query.
  - Graph-based knowledge retrieval.
  - Support for downstream medical AI applications.

## Application Scenarios

MedKG-Core can be used in the following scenarios:

- Clinical knowledge graph construction
- Medical terminology and ontology management
- Disease, symptom, drug, examination, and treatment relation modeling
- Medical record annotation and structuring
- Medical AI training data preparation
- Clinical decision support knowledge base construction
- Medical data governance and standardization
- Graph-based medical information retrieval
